Tapis de Sierpinski

Description

C’est une fractale générée à partir d’un carré de base, autour duquel on ajoute 8 carrés disposés sur les bords d’une grille de 3x3.

Dans le code, les carrés de plus petite taille sont ajoutés récursivement aux carrés existants. Le paramètre passé à la fonction de dessin correspond au nombre de division à effectuer. Sur l’illustration il vaut 4.
